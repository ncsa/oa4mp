Release notes for the OA4MP mutable version
‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾
This is not a release but will install the most recent
development branch, termed mutable, for OA4MP. Generally
this is only of interest to, for instance, developers
who are testing specific features. It should not
be installed for production purposes as there is not
guarantee that it is wholly stable.
