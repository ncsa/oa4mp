OAuth for Many People

OAuth for Many People (OA4MP) provides an OAuth 2 compliant REST web interface providing both tokens (WLCG, SciTokens and others) as well as X.509 certificates to science gateways. OA4MP has been in production since 2011 and provides a stable, mature and full-featured application.
